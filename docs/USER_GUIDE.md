## 🚀 Начало работы

### Установка

#### Desktop (Windows/Mac/Linux)

1. Скачайте последнюю версию с [GitHub Releases](https://github.com/libertyreach/messenger/releases)
2. Установите приложение
3. Запустите Liberty Reach

#### Mobile (Android/iOS)

1. Откройте Google Play / App Store
2. Найдите "Liberty Reach"
3. Установите приложение

#### Self-Hosting

См. [SELF_HOSTING.md](SELF_HOSTING.md)

## 📝 Регистрация

1. Откройте приложение
2. Нажмите "Создать аккаунт"
3. Введите имя пользователя
4. Сохраните мнемоническую фразу (12 слов)
5. Готово!

> ⚠️ **Важно:** Сохраните мнемоническую фразу в безопасном месте. Без неё невозможно восстановить доступ!

## 💬 Чаты

### Создание чата

1. Нажмите "+" в списке чатов
2. Выберите контакт
3. Начните общение

### Групповые чаты

1. Нажмите "+" → "Создать группу"
2. Введите название группы
3. Добавьте участников (макс. 1000)
4. Нажмите "Создать"

### Каналы

1. Нажмите "+" → "Создать канал"
2. Введите название канала
3. Выберите тип (публичный/приватный)
4. Пригласите подписчиков

---

## 📞 Звонки

### Аудио звонки

1. Откройте чат
2. Нажмите иконку 📞
3. Дождитесь ответа

### Видео звонки

1. Откройте чат
2. Нажмите иконку 📹
3. Разрешите доступ к камере

### Рация (Push-to-Talk)

1. Зажмите кнопку PTT
2. Говорите
3. Отпустите для прослушивания

## 🌐 AI Перевод

### Автоматический перевод чатов

1. Откройте настройки чата
2. Включите "Авто-перевод"
3. Выберите целевой язык

### Перевод звонков

1. Начните звонок
2. Включите "AI Перевод"
3. Выберите языки

### Субтитры

1. Начните видео звонок
2. Включите "Субтитры"
3. Выберите язык субтитров

## 💰 Web3 Интеграции

### Подключение MetaMask

1. Откройте "Кошелёк"
2. Нажмите "Подключить MetaMask"
3. Разрешите подключение

### Обмен токенов (0x Protocol)

1. Откройте "Обмен"
2. Выберите токены
3. Введите сумму
4. Нажмите "Обменять"

### P2P Сделки

1. Откройте "P2P"
2. Создайте сделку
3. Укажите продавца и сумму
4. Оплатите в смарт-контракт
5. После получения товара нажмите "Подтвердить"

## 📲 Миграция из Telegram/WhatsApp

### Из Telegram

1. Экспортируйте данные в Telegram Desktop:
   - Настройки → Продвинутые → Экспорт данных
2. Откройте Liberty Reach
3. Нажмите "Импорт" → "Telegram"
4. Выберите файл `result.json`
5. Дождитесь завершения

### Из WhatsApp

1. Экспортируйте чат в WhatsApp:
   - Чат → Ещё → Экспорт чата
2. Откройте Liberty Reach
3. Нажмите "Импорт" → "WhatsApp"
4. Выберите файл `.txt`
5. Дождитесь завершения

## ❓ FAQ

**Q: Это бесплатно?**  
A: Да, все функции бесплатны. Комиссия только на Web3 операциях (0.5-3%).

**Q: Безопасно ли это?**  
A: Да, используется post-quantum шифрование Kyber1024.

**Q: Можно ли использовать без интернета?**  
A: Чаты работают в локальной сети через P2P.
